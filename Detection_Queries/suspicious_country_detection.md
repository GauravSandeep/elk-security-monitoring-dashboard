# Suspicious Country Detection

## Objective
Identify VPN connections originating from unusual or unexpected geographic locations.

---

## Detection Logic
Analyze source country information to detect:
- Unexpected geographic access
- High-risk connection regions
- Abnormal connection locations

---

## Investigation Focus
- Source country trends
- Geographic anomalies
- Unexpected remote access activity
- High-frequency country activity

---

## Example Fields Used
- Source_Country
- Source_ip
- UserName

---

## Why It Matters
Unexpected geographic access may indicate compromised credentials or unauthorized VPN usage.

Geographic analysis helps SOC analysts identify suspicious remote access attempts.

---

## Skills Demonstrated
- Geographic analysis
- SIEM monitoring
- Threat detection
- Kibana dashboard analysis

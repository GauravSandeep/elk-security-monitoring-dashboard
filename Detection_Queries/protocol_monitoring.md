# Protocol Monitoring

## Objective
Monitor VPN communication protocols used during connection activity.

---

## Detection Logic
Analyze protocol usage to identify:
- Common VPN protocols
- Unusual protocol activity
- Unexpected communication behavior

---

## Investigation Focus
- Protocol frequency
- Abnormal protocol usage
- Traffic analysis
- Connection behavior

---

## Example Fields Used

- port

---

## Why It Matters
Monitoring protocol activity helps SOC analysts identify suspicious network communication and abnormal traffic patterns.

# Dashboard Overview

## Dashboard Name
VPN Security Monitoring Dashboard

---

## Objective
Visualize VPN connection activity using Kibana dashboards for security monitoring and threat analysis.

---

## Included Visualizations
- VPN connections over time
- Top source countries
- Most active source IPs
- Protocol usage analysis
- VPN port monitoring

---

## Monitoring Goals
- Detect suspicious VPN activity
- Analyze connection trends
- Monitor geographic access patterns
- Investigate abnormal network behavior

---

## Tools Used
- Kibana
- Elasticsearch
- Elastic Stack

---

## Skills Demonstrated
- Dashboard creation
- SIEM monitoring
- Security visualization
- Threat analysis
